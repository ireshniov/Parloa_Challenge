# Parloa Challenge

## How to run

### Start the app in dev mode

```shell
npm run start:dev
```

### Start the app in prod mode

```shell
npm run build:dist
npm run start
```

### Running tests

```shell
npm run test
```

### Coverage

```shell
npm run coverage
```

## Utils

### Lint

```shell
npm run lint
```

### Format

```shell
npm run format
```

### Prettier

```shell
npm run prettier
```
